## 1) 복합대입 연산자

<img src='images/2021-08-14-22-16-21.png' />

- JAVA

  ```JAVA
    public class OperationEx2 {

        public static void main(String[] args) {

            int num1 = 10;

            System.out.println("num1 +=1 = ");
            System.out.println(num1 +=1);

            System.out.println("num1 %=10 = ");
            System.out.println(num1 %= 10);

        }
    }
  ```

- Result - Terminal
  ```TEXT
    num1 +=1 =
    11
    num1 %=10 =
    1
  ```
